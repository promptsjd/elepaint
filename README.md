# elepaint

**Generate beautiful color palettes using real color theory — in one click.**

elepaint applies the **60:30:10 rule** — the same principle used by interior designers and brand strategists — to produce color schemes that are visually balanced by construction, not by accident. Every color is generated in **OKLCH**, a perceptually uniform color space, so combinations always feel intentional rather than random.

## How it works

Each palette gives you three colors with a clear role:

| Swatch | Role | Use it for |
|---|---|---|
| Top | Primary (60%) | Backgrounds, large fills, page canvas |
| Middle | Secondary (30%) | Cards, surfaces, sidebars, body text regions |
| Bottom | Accent (10%) | Buttons, links, highlights, icons |

Click any swatch to **copy its hex code** to your clipboard instantly. Hover first to preview it.

---

## Harmony modes

| Mode | Description |
|---|---|
| **Random** | Picks a harmony type automatically |
| **Analogous** | Neighboring hues — serene and cohesive |
| **Complementary** | Opposite hues — high contrast |
| **Triadic** | Three evenly spaced hues — vibrant and balanced |
| **Split-Complementary** | Dynamic contrast without full opposition |
| **Monochromatic** | One hue across three depths of lightness |

## Development

To run the extension locally:

1. Open Chrome and go to `chrome://extensions`
2. Enable **Developer mode** (toggle, top-right)
3. Click **Load unpacked** and select this folder
4. Click the elepaint icon in the toolbar to open the popup

After editing any file, go back to `chrome://extensions` and click the **refresh icon** on the elepaint card to reload.
